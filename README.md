# java-cli-example
Java CLI example

## Development

1. Initialize a project.
    ```bash
    $ gradle init --type java-application
    ```
1. Test.
    ```bash
    $ gradlew test
    ```
1. Build JAR.
    ```bash
    $ gradlew jar
    ```
1. Run.
    ```bash
    $ java -jar ./build/libs/java-cli-example-*.jar
    ```
